INTRODUCTION TO FRONT-END DEVELOPMENT

Sept 29 | 2 min read

What is front-end development?

front-end web development is also known as client-side development. In this the developer scripts Hyper Text Markup Language , Cascading Style Sheets and JavaScript code for a webpage so the users can directly interact with them. There are significant differences between back-end development,front-end development, and full stack development. A front end developer works on the aspects of an website/app that users get to interact with. Whereas a back-end developer handles the work behind the website , say the databases the connectivity and processing of data. Ultimately, a full-stack developer does the job of both, He's is someone who wprks on the whole design process from start to finish.

Tools used for front-end development

The major languages for the front-end development are HTML, CSS and Javascript. Jquery is also used for the front-end development but it is said to be going out of style but many legacy projects still use this JavaScript library. Front-end development extensively deals with responsive design, typography, color theory, layout and grid system. Lets see how these languages are used in the front-end development.

HTML

HTML stand for Hypertext Markup Language. Hypertext means a text which has link embedded in it called hyperlinks. So whenever users clicks on that text they gets redirected to another web-page.A markup language also means that the text can be converted into image, link, table and other formats. The structure and a look of the web page is written in HTML code.

CSS

CSS stands for Cascading Style Sheets. It is used for describing the presentation of web pages written in a markup languages like HTML. So basically it deals with the presentation aspect of the site and gives your site its own unique look. CSS handles the look and feel part of a web page. Using css you can control color of text, fonts of a text, paragram spacing, background images, variations in display for different devices and screen sizes and many more effects thus making websites presentable to the users.

JavaScript

JavaScript is a client-side scripting language. JavaScript is used to enhance the interaction of a user with webpage.So basically it is used to transform a static HTML page into a dynamic interface. JavaScript code uses document object model (DOM) to manipulate a web page according to events, such as on click. Most of the websites use it for client-side page behavior. All major web browsers have a dedicated JavaScript engine to execute JavaScript. In addition to that there are many modern front-end development frameworks are available like Angula.js, React.js, React Native, etc which makes a development a lot more faster and allow one to build modular and reactive components.
